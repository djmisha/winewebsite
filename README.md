# Wine Website



## Development

1. In terminal, go to the folder with files and run gulp command.

-   cd (directory)
-   npm install   
-   gulp watch

2. Fire up browser and code editor.

## Hosting

-   Site is hosted on Netlify.
-   Site deploys automatically from the "Main" repository branch.
